# Input

Configurated input

## Props

| Name            | Type                | Default         | Description|
|:-----|:-----|:-----|:-----|
| `wide` | `Boolean` | `false` | Make button full width of container


## Example

```jsx
import React from 'react'
import { Input, Row } from 'pattern-library/atoms'

export default () => (
  <Row marginBetween={10} justifyContent="spaceBetween">
    <Input>Default</Input>
  </Row>
)
```
